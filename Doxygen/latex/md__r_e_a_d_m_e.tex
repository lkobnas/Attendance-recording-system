\doxysection*{Punch\+IN -\/ A Smart Attendance Recording System }

~\newline




 

\doxysubsection*{About the Project }

Tired of taking attendance by hand? This project will surely help you out! Punch\+IN is a real-\/time raspberry Pi project which aims to simplify the process of taking attendance by introducing a smart device. It not only saves time but also ensures accuracy and reduces the chance of manual errors. This system eliminates the need for manual signatures and allows students to mark their attendance in just a few seconds. ~\newline

\doxysubsection*{~\newline
 Features }

\doxysubsubsection*{1. Take attendance with three methods }


\begin{DoxyItemize}
\item \mbox{\hyperlink{class_r_f_i_d}{RFID}} / NFC
\item Facial recognition
\item Fingerprint
\end{DoxyItemize}

Maximum Flexibility!

~\newline


\doxysubsubsection*{2. Friendly UI design }

Intuitive UI design with three operation modes\+:


\begin{DoxyItemize}
\item Attendance Taking mode
\item User Registration mode
\item Class Scheduling mode
\end{DoxyItemize}

~\newline


\doxysubsubsection*{3. \mbox{\hyperlink{class_email}{Email}} notification }


\begin{DoxyItemize}
\item A confirmation email will be sent out automatically when a student has punched in
\item A reminder email will be sent to students who registered in a class but did not show up on time
\end{DoxyItemize}

~\newline


\doxysubsubsection*{4. Real-\/time attendance record }

All scheduled classes and attendance records are saved into the local database. \mbox{\hyperlink{struct_course}{Course}} instructors can review the real-\/time attendance record in the front panel at any time.
