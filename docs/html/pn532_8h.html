int PN532_WriteGpio ( PN532 *  pn532,
uint8_t *  pins_state 
)

: Write the GPIO states.

Parameters
pins_statepin state buffer (2 bytes) to write. no need to read pin states before write with the param pin_state P3 = pin_state[0], P7 = pin_state[1] bits: P3[0] = P30, P7[0] = 0, P3[1] = P31, P7[1] = P71, P3[2] = P32, P7[2] = P72, P3[3] = P33, P7[3] = nu, P3[4] = P34, P7[4] = nu, P3[5] = P35, P7[5] = nu, P3[6] = nu, P7[6] = nu, P3[7] = Val, P7[7] = Val, For each port that is validated (bit Val = 1), all the bits are applied simultaneously. It is not possible for example to modify the state of the port P32 without applying a value to the ports P30, P31, P33, P34 and P35.
