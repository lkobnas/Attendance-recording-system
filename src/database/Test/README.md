# Database Testing

## Install Google Test and Cmake
```
sudo apt-get install libgtest-dev -y
sudo apt-get install cmake
```
```
cd /usr/src/gtest
sudo make
```

## Running test

Create and navigate to the build folder
```
mkdir build && cd build
```

Cmake
```
camke ..
```

Build 
```
make
```

Run
```
./main
```
